# Engine

The `Engine` type is a deterministic, faithful simulation of the TETR.IO game engine. It is driven frame-by-frame by the game loop but is also usable standalone for analysis, solving, and replay playback.

```rust
use triangle_rs::engine::{Engine, EngineInitParams, EngineEvent, LockResult, ResCache, EngineSnapshot};
```

## Creating an engine

In most cases the engine is created for you by `create_engine()` inside `Game::new()`. For standalone use:

```rust
let params = EngineInitParams {
    queue: QueueInitParams { seed: 0, bag_type: BagType::SevenBag, .. },
    board: BoardInitParams { width: 10, height: 20, buffer: 20 },
    kick_table: "SRS+".to_string(),
    options: GameOptions { .. },
    gravity: IncreasableValue { value: 0.02, increase: 0.0, margin_time: 0 },
    garbage: GarbageQueueInitParams { .. },
    handling: HandlingOptions { arr: 0.0, das: 133.0, dcd: 0.0, sdf: 41.0, .. },
    pc: None,
    b2b: B2bOptions { chaining: true, charging: None },
    multiplayer: None,
    misc: MiscOptions { .. },
};

let engine = Engine::new(params);
```

## Ticking

```rust
pub fn tick(&mut self, frames: &[ReplayFrame]) -> ResCache
```

Advance the engine by the given replay frames (keypresses + timestamps). Returns a `ResCache` summarising changes since the previous tick.

```rust
pub struct ResCache {
    pub pieces: i32,              // pieces placed this tick
    pub garbage_sent: Vec<i32>,   // garbage line counts sent
    pub garbage_received: Vec<OutgoingGarbage>,
    pub keys: Vec<String>,        // key names that were pressed
    pub last_lock: f64,           // frame of last lock
}
```

To advance with no input (simulate gravity / garbage tanking):

```rust
let result = engine.tick(&[]);
```

## Events

After each `tick()` call, `engine.events` contains a list of `EngineEvent` values emitted during that tick. Read them before the next tick — they are cleared on the next call.

```rust
pub enum EngineEvent {
    QueueAdd(Vec<Mino>),
    GarbageReceive { iid: i32, amount: i32, original_amount: i32 },
    GarbageConfirm { iid: i32, gameid: i32, frame: i32 },
    GarbageTank    { iid: i32, column: usize, amount: i32, size: usize },
    GarbageCancel  { iid: i32, amount: i32, size: usize },
    FallingNew  { piece: Mino, is_hold: bool },
    FallingLockPre,
    FallingLock(LockResult),
}
```

### `LockResult`

Attached to `FallingLock`:

```rust
pub struct LockResult {
    pub mino: Mino,
    pub garbage_cleared: usize,
    pub lines: usize,
    pub spin: SpinType,         // None, Mini, Full
    pub raw_garbage: Vec<i32>,  // before reductions
    pub garbage: Vec<i32>,      // after reductions
    pub surge: i32,
    pub stats: EngineStats,
    pub garbage_added: Option<Vec<OutgoingGarbage>>,
    pub topout: bool,
    pub piece_time: f64,
    pub key_presses: Vec<String>,
}
```

Example — detect line clears:

```rust
for event in &engine.events {
    if let EngineEvent::FallingLock(lock) = event {
        println!("Cleared {} lines with spin {:?}", lock.lines, lock.spin);
        if lock.topout {
            println!("topped out!");
        }
    }
}
```

## Board

```rust
pub struct Board {
    pub state: Vec<Vec<Option<Tile>>>,  // [row][col], row 0 is bottom
    pub height: usize,
    pub width: usize,
    pub buffer: usize,  // hidden rows above the playfield
}
```

### `Tile`

```rust
pub struct Tile {
    pub mino: Mino,
    pub connections: u8,  // bitmask of filled neighbours
}
```

Connection bitmask constants:

| Constant      | Value      | Meaning                |
| ------------- | ---------- | ---------------------- |
| `CONN_TOP`    | `0b1000`   | Neighbour above        |
| `CONN_RIGHT`  | `0b0100`   | Neighbour to the right |
| `CONN_BOTTOM` | `0b0010`   | Neighbour below        |
| `CONN_LEFT`   | `0b0001`   | Neighbour to the left  |
| `CONN_CORNER` | `0b1_0000` | Corner decoration      |
| `CONN_ALL`    | `0b1111`   | All four neighbours    |

### Board methods

```rust
// True if (x, y) is occupied (x = column, y = row from bottom).
pub fn occupied(&self, x: i32, y: i32) -> bool

// Add blocks to the board.
pub fn add(&mut self, blocks: &[(Tile, i32, i32)])

// Clear completed lines.  Returns how many lines and garbage rows were cleared.
pub fn clear_lines(&mut self) -> ClearResult

// True if the board is completely empty.
pub fn perfect_clear(&self) -> bool

// Insert garbage rows.
pub fn insert_garbage(&mut self, params: InsertGarbageParams)

// Full height (height + buffer).
pub fn full_height(&self) -> usize

// Reset to empty.
pub fn reset(&mut self)
```

Reading cells:

```rust
for row in 0..engine.board.height {
    for col in 0..engine.board.width {
        match &engine.board.state[row][col] {
            Some(tile) => print!("{:?} ", tile.mino),
            None => print!("  "),
        }
    }
    println!();
}
```

## Queue

`engine.queue` exposes the upcoming piece sequence.

```rust
use triangle_rs::engine::queue::types::Mino;

// The piece currently falling (not from the queue — it has already been dequeued).
let current: Mino = engine.falling.symbol;

// Peek at all pieces still in the visible queue.
let next: &[Mino] = engine.queue.as_slice();
```

### `Mino` enum

```
I  O  T  S  Z  J  L  Garbage  Empty
```

## Snapshots

Take a full deterministic snapshot:

```rust
let snap: EngineSnapshot = engine.snapshot(false);

// Restore
engine.from_snapshot(&snap, false);
```

`is_undo_redo` — pass `true` only when integrating with the engine's own undo/redo stack.

`EngineSnapshot` contains full board state, queue state, garbage queue, statistics, and input state. It is `Clone` and can be stored freely.

## Reset

Resets the engine to its initial state (same seed, same options):

```rust
engine.reset();
```

## `HandlingOptions` (engine-level)

Identical structure to the top-level `Handling`, but used internally by the engine simulation. Obtained from `EngineInitParams::handling`. See [Client.md](Client.md) for field descriptions.
